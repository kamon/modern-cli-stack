# Why Your Terminal Is Slow — And How to Speed It Up

You probably type `cd` more than any other command. You probably hit
`↑` to find that one `git` command from yesterday. You probably open
your file manager to find the project you were working on five minutes
ago.

This PDF is about thirteen focused tools, each doing one thing well. After
installing them, you'll navigate to old projects with two characters,
search code instantly, and read git diffs that don't hurt your eyes.

## Who this is for

You're new to the command line, or you've been using it for years and
suspect it could be faster.

The install script takes one command. Each tool is independent — pick
the ones that solve a problem you actually have.

## How to use this PDF

The first three sections (Mental Models, The Install Script, The 13
Tools) are the core. Read them in order. The remaining sections
(roadmap, appendices) are reference — come back when you need them.

Uninstall instructions are in Appendix D (or run
`./install-cli-stack.sh --uninstall` for an interactive
removal). Nothing here is irreversible.
