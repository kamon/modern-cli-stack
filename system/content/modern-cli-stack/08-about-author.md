## About the author

Kamon Ayeva is a Python consultant and author focused on practical,
production-grade development workflows. He is the co-author of
*Mastering Python Design Patterns* (Packt), and writes short, regularly
updated guides on modern Python development.

Kamon's work focuses on the gap between documentation and practice:
the build systems, the dependency models, and the editorial patterns
that make a codebase safe to change.
