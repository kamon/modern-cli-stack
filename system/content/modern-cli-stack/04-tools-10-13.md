# The 13 Tools (continued)

## 10. delta — Beautiful Git Diffs

**Side-by-side, syntax-highlighted `git diff` you actually want to read.**

Syntax-highlighting pager for `git diff`, `git log`, `git show`. Replaces the
wall-of-red-and-green with aligned, colored, line-numbered output.

### Install

```bash
# macOS
brew install git-delta

# Linux / WSL
# See https://github.com/dandavison/delta for installer
```

Add to `~/.gitconfig`:

```ini
[core]
    pager = delta

[interactive]
    diffFilter = delta --color-only

[delta]
    navigate = true
    line-numbers = true
    side-by-side = true
```

### Before / After

- **Before:** `git diff` — red/green blocks you scroll through.
- **After:** `git diff` — aligned, syntax-highlighted, line-numbered,
  side-by-side.

**Try this:** `git log -p` — beautifully rendered commit history with
diffs.

## 11. tldr — Simplified Man Pages

**Man pages, but for humans.**

Community-maintained, simplified command examples. Skips the 40-option man
page, gives you 5-10 practical examples.

### Install

```bash
# macOS
brew install tldr

# Linux / WSL
sudo apt install tldr
```

### Before / After

- **Before:** `man tar` — 200 lines, exits with `:q` confusion.
- **After:** `tldr tar` — 10 common examples, color-coded.

**Try this:** `tldr ffmpeg`, `tldr kubectl`, `tldr jq`. Run
`tldr --update` weekly.

## 12. atuin — Magical Shell History

**Searchable, synced, deduplicated shell history across machines.**

Replaces your shell history with a SQLite-backed, searchable, optionally
encrypted-and-synced history.

### Install

```bash
# macOS
brew install atuin

# Linux / WSL
curl --proto '=https' --tlsv1.2 -sSf https://setup.atuin.sh | sh
```

Run `atuin import` to import existing history, then `atuin login` for sync
(optional).

### Before / After

- **Before:** `Ctrl+R` searches current machine only, loses history across
  machines.
- **After:** `Ctrl+R` searches all machines (if synced), full-text, fuzzy,
  with date filters.

**Try this:** `Ctrl+R`, type part of an old command. Run
`atuin search --before "yesterday"` for time-based queries.

## 13. lazygit — TUI for Git

**Your terminal now has a GUI for Git.**

Full terminal UI for staging, committing, branching, merging, rebasing — all
with keyboard.

### Install

```bash
# macOS
brew install lazygit

# Linux / WSL
sudo apt install lazygit
```

**Try this:** `lazygit` in any repo -> press `?` for help.

### Bonus: `gh` — GitHub from CLI

If you work with GitHub, also install `gh` (GitHub CLI):

```bash
brew install gh          # macOS
sudo apt install gh      # Linux / WSL
```

Useful commands: `gh repo view`, `gh pr create`, `gh issue list`,
`gh workflow run`.
